「バースト! 人間行動を支配するパターン」読了

Aug 7th, 2012

感想

今回は纏めきれなかったのでネタでお茶を濁させて下さい…．

以下Twitterで書いた感想(適当)

バラバシの新刊「バースト 人間行動を支配するパターン」を読んだけど，いろんな意味で難解な本だった．この本で提唱されるバーストと呼ばれる現象を16世紀ハンガリーの十字軍の歴史と絡めながら語られる様は正に圧巻というか，1回通して読んだだけじゃ理解できないです…

— やぐ (@yag_ays) August 6, 2012

まあバーストは基本的にべき乗則やランダムウォーク，レヴィ飛行とかと一緒に説明されて実例とともに分かりやすく解説されるんだけど，筋はハッキリしてるけど系統立てて話されないとうか，そもそもハンガリーの十字軍の話と交互に出てきて結構飛び飛びなんだよなぁ．

— やぐ (@yag_ays) August 6, 2012

バラバシの研究遍歴とともに語られるバーストという現象の話は非常に面白い．あいだに入ってくる16世紀十字軍の話はきちんと中盤で直接的なつながりが出てくるし物語自体もこれだけで十分楽しめるレベルだと思う．ただし手っ取り早く流し読みで要点抑えるとかしづらいので読むときは覚悟がいるかも

— やぐ (@yag_ays) August 6, 2012

結局のところ予測ってなんだろう．ランダムってなんだろう．外れ値ってどう扱えばいいんだろう．じゃあ80％で予測が当たるって何を意味するんだろう．とか考えだすと色々と止まらなくなる

— やぐ (@yag_ays) August 6, 2012
